# WGU Brand and Design System: Do and Don't Checklist

A fast-reference card. Use before shipping any asset.

## Brand fundamentals

**Do**

- Lead with the student. The student is the protagonist.
- Speak in second person ("you") to students.
- Use evidence-backed claims with specific numbers.
- Use active voice and strong verbs.
- Match copy length to format (billboard 7 words max, 15s video 25 words max, paid social 27 character headline).

**Don't**

- Use owl puns or owl-related wordplay.
- Use emoji in brand collateral.
- Hedge ("may", "could potentially").
- Refer to students in third person when speaking to them.
- Use buzzwords ("revolutionary", "best in class").

## Color

**Do**

- Include WGU Blue (#002855) on every designed asset.
- Use exact HEX values from the palette.
- Pair Lime Green sparingly. One pop per layout.
- Use approved gradients only (Midnight, Medium Blue, Protection).

**Don't**

- Tint, lighten, darken, or eyeball any brand color.
- Adjust opacity on brand colors.
- Use black for body text (use WGU Blue).
- Pair Midnight bg with Grey text, Lime bg with Medium or Sky Blue text, or White bg with Sky Blue or Lime text.

## Typography

**Do**

- Use Futura PT (Arial fallback) for body and headlines.
- Use Rocky (Newsreader substitute) only for isolated infographic numerals.
- Use Program Nar OT Black (Oswald substitute) only for Education for the People campaign headlines.
- Set body text at 18px / 26px line height on web.
- Flag font substitutions for production work.

**Don't**

- Mix Futura PT with another sans-serif.
- Set entire paragraphs in ALL CAPS.
- Use Rocky or Oswald in body copy.
- Stretch, condense, or skew letterforms.

## Logos and marks

**Do**

- Use the university logo for public-facing comms.
- Use the owl icon alone only for student, alumni, or employee audiences.
- Respect minimum sizes (52px web, 0.725 inch print).
- Respect clear space (one-half X-height for master mark, one-third icon height for owl).
- Pull official assets from Canto.

**Don't**

- Combine the wordmark and the owl icon into a custom lockup.
- Recolor, rotate, stretch, or otherwise modify any mark.
- Use the Academic Seal outside official documents without approval.
- Source logos from web screenshots or third-party sites.

## Iconography

**Do**

- Use Simple icons (line-based, one-tone) for web UI navigation.
- Use Illustrated icons (two-tone or color background) only in marketing collateral.
- Substitute Lucide for Simple icons when the official set is not available.
- Color icons with palette values only.

**Don't**

- Modify, stretch, recolor, or overlap icons.
- Create new icons. Brand Governance owns the library.
- Use Illustrated icons in product UI.
- Use one-tone Simple icons outside web nav.

## Photography

**Do**

- Use authentic, vibrant, warm-toned student photography.
- Allow real grit. Avoid heavy retouching.
- Apply gradient overlays so headlines stay legible.
- Pull from Canto.

**Don't**

- Use generic stock photography.
- Tint photos outside approved gradient overlays.
- Imply a traditional brick-and-mortar campus.
- Convey critical information through visual cues alone.

## Patterns

**Do**

- Use one pattern per asset.
- Match pattern theme to content (Pathways for programs, Network for partners).
- Place patterns at low contrast under content panels.

**Don't**

- Stack two patterns on the same asset.
- Modify, recolor, or scale patterns into banding.
- Place copy directly on a pattern without a content panel.

## Layout and spacing

**Do**

- Use the 4px base grid and the spacing scale.
- Allow generous whitespace around the logo and primary content.
- Keep page max-width 1200 to 1280px on desktop.
- Top-align card titles within a row.

**Don't**

- Use spacing values outside the scale.
- Crowd typography against borders or imagery.
- Center long body paragraphs.
- Reduce the logo below minimum size to fit a tight space.

## Components

**Do**

- One primary button per primary user task.
- 44px default button height.
- Visible focus ring on all interactive elements.
- Validate forms on blur, not on every keystroke.

**Don't**

- Use Lime Green as a button background.
- Disable submit buttons by default.
- Stack multiple primary buttons.
- Use placeholder text as the only label.

## Motion

**Do**

- Use ease-out curves (`cubic-bezier(0.2, 0, 0.2, 1)` standard).
- Keep durations short (120ms hover, 220ms state, 420ms entry).
- Use fades and gentle slides only.
- Honor `prefers-reduced-motion`.

**Don't**

- Use bounces, springs, or elastic motion.
- Animate logos.
- Use parallax on hero photography.
- Run motion longer than 500ms.

## Final pre-ship gut check

- WGU Blue (#002855) is present.
- All brand colors are exact HEX, no tints.
- Logo, seal, icons, patterns are unmodified.
- Wordmark not paired with owl icon.
- Type uses approved family for the surface.
- Copy is owl-pun-free and emoji-free.
- Focus rings are visible.
- Pulled from Canto where applicable.

If any of those is no, fix it before shipping.
