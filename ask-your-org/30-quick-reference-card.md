# WGU Quick Reference Card

A one-page cheat sheet of the most-asked values.

## Core color

- **WGU Blue:** `#002855` (required on every asset)
- **Midnight:** `#001731`
- **Medium Blue (CTA, links):** `#0070F0`
- **Sky Blue (accent):** `#46B1EF`
- **Lime Green (sparingly, one pop per layout):** `#97E152`
- **Light Blue surface:** `#EEF6F9`
- **Light Grey surface:** `#F1F1F1`
- **Grey muted:** `#A7A7A7`

## Brand gradients (web only)

- **Midnight gradient:** `linear-gradient(180deg, #001731 0%, #002855 100%)`
- **Medium Blue gradient:** `linear-gradient(135deg, #002855 0%, #0070F0 100%)`

## Type

- **Primary:** Futura PT (web substitute: Jost)
- **System fallback:** Arial
- **Numerals only:** Rocky (web substitute: Newsreader)
- **EFTP campaign only:** Program Nar OT Black (web substitute: Oswald)

## Type scale (web)

- h1: 55px / 48px / Heavy 800
- h2: 55px / 48px / Medium 500
- h3: 26px / 22px / Heavy 800
- h4: 26px / 22px / Medium 500
- Body: 18px / 26px / Book 400
- Small: 16px / 22px
- Eyebrow: 13px ALL CAPS, tracking 0.12em

## Spacing scale (4px base)

4, 8, 12, 16, 24, 32, 48, 64, 96.

## Radius

0, 4, 8 (default), 16 (feature), 999 (pill).

## Shadow

- `--shadow-sm`: `0 1px 2px rgba(0, 23, 49, 0.08)`
- `--shadow-md`: `0 4px 12px rgba(0, 23, 49, 0.10)`
- `--shadow-lg`: `0 12px 32px rgba(0, 23, 49, 0.14)`
- `--shadow-focus`: `0 0 0 3px rgba(0, 112, 240, 0.35)`

## Motion

- Easing standard: `cubic-bezier(0.2, 0, 0.2, 1)`
- Easing emphasis: `cubic-bezier(0.2, 0.8, 0.2, 1)`
- Durations: 120ms / 220ms / 420ms

## Logo minimums

- 52px wide on web.
- 0.725 inch wide in print.

## Logo clear space

- Master mark: half the X-height on all four sides.
- Owl icon: one-third the icon height on all four sides.

## Copy length caps

- OOH billboard: 7 words.
- 15-second video: 25 words.
- Paid social headline: 27 characters.

## The 8 hard rules

1. WGU Blue (#002855) on every asset.
2. Never alter logos, seal, icons, patterns.
3. Never tint or adjust opacity of brand colors.
4. Wordmark always without owl icon.
5. Owl icon alone only for student / alumni / employee audiences.
6. Academic Seal restricted to official documents.
7. No owl puns, no emoji in brand collateral.
8. Type rules: Futura PT (or Arial), Rocky for isolated numerals only, Program Nar OT Black for EFTP campaign headlines only.
