# WGU Marketing Site — UI Kit

A high-fidelity recreation of WGU.edu's marketing surface using the brand system in this project. This kit is **not** built from the live wgu.edu codebase (none was provided); it is built from the brand guide's visual foundations + Evergreen template + public-facing copy conventions. Treat it as "what a WGU marketing page SHOULD look like" rather than a 1:1 mirror.

## Files
- `index.html` — interactive click-thru marketing homepage
- `Header.jsx` — top nav with primary/secondary CTAs
- `Hero.jsx` — midnight-gradient hero with headline + form
- `ProgramGrid.jsx` — schools/programs browser
- `StatBand.jsx` — infographic stat row
- `Testimonial.jsx` — student quote section
- `CtaBand.jsx` — mid-page "Apply" CTA
- `Footer.jsx` — midnight footer

## Flags
- Photography is replaced with solid colors / patterns (no student cutouts in this project).
- Icons are Lucide (see root README ICONOGRAPHY).
- Fonts are Jost (Futura PT substitute) + Newsreader (Rocky substitute).
