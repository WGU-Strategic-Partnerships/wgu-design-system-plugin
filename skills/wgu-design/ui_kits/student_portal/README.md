# WGU Student Portal — UI Kit

This kit is **intentionally minimal** — the WGU student portal (My WGU / Workday Student / Navigate) was not provided as a codebase, Figma, or screenshots. Rather than invent one, this directory contains only a single "signed-in dashboard" stub built from brand principles. Treat it as a starting point, not a recreation.

If you are iterating on a portal design:
- Pull real screenshots / source from `my.wgu.edu` or the internal Workday/Navigate repo.
- Ask the user for access or to attach the codebase via Import.
- Use the marketing-site UI kit's components (button, form, card) as a starting visual language — the portal shares those primitives.
